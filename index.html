Firebase Setup Required
// ONE-TIME CONFIGURATION
⚠ Add your free Firebase credentials to this file before it will work. Takes ~5 minutes.
1. Go to console.firebase.google.com → sign in with Google
2. Create a project → left sidebar → Firestore Database → Create database → Start in test mode → Enable
3. Left sidebar → Authentication → Get started → Sign-in method → Enable Google → Save
4. ⚙ gear → General → scroll to "Your apps" → click </> → register → copy the config values
5. Open this file in a text editor, find FIREBASE_CONFIG at the top, paste your values in. Save.
6. Drag the file's folder onto netlify.com to get a shareable link. Then add your Netlify URL to Firebase → Authentication → Settings → Authorized domains.
Once deployed, friends just open your Netlify URL — no setup needed on their end.
